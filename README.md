## Coding Quiq

## Description
This application lets user take javascript quiz with multiple choice questions which are timed and saves Highscore.For every wrong answer 10 secs and 1 point are deducted.


## Built With
* HTML
* CSS
* JavaScript
* Web API's
* Events
* Client-Side Storage
